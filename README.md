# 🖥️ Interactive macOS Portfolio

A premium, interactive web-based portfolio mirroring a macOS desktop environment. Built with Next.js, React, and TypeScript, it allows users to open and interact with multiple windowed applications, change wallpapers, check the weather, play games, and explore projects in a simulated OS interface.

## ✨ Features

- **Window Management System**: Open, drag, minimize, maximize, and stack multiple applications.
- **Boot, Login, and Sleep States**: Beautiful simulated boot sequence, password login screen, and sleep/power state controls.
- **Control Center & Top Menu**: A fully functional top status bar with a clock, controls for Wi-Fi, theme, volume, and language translation.
- **Built-in Interactive Apps**:
  - **Finder**: Browse through folders containing resume info, projects, and contact cards.
  - **Safari**: Interactive mock browser.
  - **Snake Game**: Playable retro snake game.
  - **Notes & TextEdit**: View text content and write-ups.
  - **FaceTime & Music**: Mock media applications.
  - **Spotlight Search**: Press `Cmd + Space` or click to search files and launch apps.
  - **Terminal**: Run basic shell commands to browse files.
- **Multilingual Support**: Fully localized in English (EN) and Spanish (ES).

## 🛠️ Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **UI & Logic**: React, TypeScript, TailwindCSS
- **State Management**: Zustand / Custom Store hooks

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm, yarn, or pnpm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
